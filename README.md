# SuperCoin-BlockChain

Flipkart SuperCoin loyalty points on blockchain

## Dependencies

- Node 16.14.1
- Truffle suite
  - Truffle 5.11.2
  - Ganache 7.9.0
  - Solidity 0.8.21
  - Web3.js 1.10.0

install Truffle Suite

```bash
    npm install -g truffle
```

## Setup instructions

Steps to run on local truffle develop

```bash
cd SuperCoin-BlockChain/
truffle compile
truffle develop
truffle migrate
```
